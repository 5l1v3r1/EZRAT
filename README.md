## Features :
- [x] Client Reverse TCP
- [x] Client Hiding
- [x] MultiClient
- [x] Get informations from client
- [x] File browsing client
- [x] File transfert (only for small file ~10MB)
- [x] Disconnect server correctly
- [x] Download upload file
- [ ] Startup
- [x] Process viewer
- [ ] Remote desktop
- [x] ScreenShot
- [ ] Camera stream
- [ ] Sound play
- [ ] System info in details
